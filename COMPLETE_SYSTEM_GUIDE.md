## ğŸ”§ Troubleshooting

### Encoding Issues

**Problem**: Perplexity queries still scrambled

**Solution**:
1. Restart Cursor (critical!)
2. Test with: "test simple query"
3. If still broken, see `PERPLEXITY_ENCODING_ISSUE.md`

### Space Creation Fails

**Problem**: Browser automation can't create Space

**Solution**:
1. Make sure logged in to Perplexity.ai
2. Create Space manually
3. Register: `Set-ProjectSpace -SpaceUrl "..."`
4. System works normally with manual Space

### Wrong Space Used

**Problem**: Searches go to wrong project's Space

**Solution**:
```powershell
# Check which Space is configured
Get-Content ".cursor\mcp-local.json"

# Reconfigure if needed
Get-ProjectSpace  # Verify registry
Configure-ProjectMCP.ps1 -ProjectPath "." -SpaceUrl "correct-url"
```
